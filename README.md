# 🎵 Lyrics Generation – NLP Project

This project implements a **Natural Language Processing (NLP) system for generating song lyrics** using machine learning techniques and a dataset combining **Spotify metadata** with **Genius lyrics links**.

Developed as part of a **Master’s course project at the German International University (GIU)**.

---

## 📌 Overview
The goal of this project was to design and evaluate a system capable of **generating coherent song lyrics** by learning patterns from existing lyrics datasets.

The project included:
- 📄 Defining project **objectives, scope, and success criteria** in a structured thesis report  
- 🧩 Documenting **functional requirements, data requirements, assumptions, and constraints**  
- 🧹 Preparing and **preprocessing lyrics datasets** for NLP modeling  
- 📊 Evaluating generated lyrics against criteria such as **coherence and contextual relevance**
